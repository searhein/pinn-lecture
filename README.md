# Neural Networks and Physics-Informed Neural Networks

This repository contains two Jupyter notebooks accompanying the presentation *Can a Neural Network Learn the Laws of Nature?* The notebooks are designed for workshop participants and can be run directly in the browser via Binder.

## Notebooks

- `nn-notebook.ipynb`  
  A small neural-network example that learns a sine curve from sample data.

- `pinn-notebook.ipynb`  
  A physics-informed neural network (PINN) example for an RC circuit.

## Run Online with Binder

You can open the notebooks directly in your browser:

- Neural network example:  
  https://mybinder.org/v2/gh/searhein/pinn-lecture/HEAD?labpath=nn-notebook.ipynb

- Physics-informed neural network example:  
  https://mybinder.org/v2/gh/searhein/pinn-lecture/HEAD?labpath=pinn-notebook.ipynb

Binder may take a short moment to start. Once it has loaded, open the notebook and run the cells from top to bottom.

## What You Can Explore

In the notebooks, you can change parameters and see how the results change. For example, you can explore:

- the amount of training data,
- the noise level,
- the size of the neural network,
- the number of training steps,
- and, in the PINN example, the balance between data and physics.
